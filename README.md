# Playwright Test Automation for SauceDemo

## Prerequisites

- Java 11+
- Maven
- Playwright (Java setup)
- IDE: IntelliJ IDEA or Eclipse

## Setup Instructions

1. Clone the repository
2. Install dependencies via Maven
3. Set up Playwright
4. Configure `config.properties` (optional)

## Running Tests

### Command Line

```bash
mvn clean install
mvn test
